Quality Dashboard
=================

View the live quality metrics and analysis reports.

The Quality Dashboard aggregates results from:

* **CodeQL** - Security and code quality analysis using MISRA C++ standards
* **Clang-Tidy** - Static analysis warnings and suggestions
* **Code Coverage** - Line, function, and branch coverage metrics
